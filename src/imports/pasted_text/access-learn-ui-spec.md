Design a modern, clean, and highly intuitive web application UI for a platform called Access Learn.

Access Learn is an AI-powered online classroom platform that removes language and accessibility barriers in education. The platform allows teachers to teach students globally while AI translates lectures, generates transcripts, and assists with assignments.

Target users:

Teachers conducting online classes

Students attending classes

Students who speak different languages

Students who are deaf or hard of hearing

Design style:

Modern SaaS dashboard

Clean, minimal, professional

Colors: deep blue, white, and light purple accents

Rounded cards, smooth shadows

Accessible typography

Responsive design for laptop screens

Core UI Screens to design:

Landing Page
Include:

Product hero section

Tagline: "Education Without Barriers"

Features overview

Call-to-action buttons (Start Teaching / Join Class)

Illustrations showing AI classroom translation

Login and Sign Up Page
Include:

Email login

Role selection (Teacher or Student)

Clean minimal form

Friendly onboarding visuals

Teacher Dashboard
Features:

Create Classroom button

List of existing classes

Class code display

Assignment manager

Class analytics (students, engagement)

Student Dashboard
Features:

Join class with class code

List of enrolled classes

Upcoming assignments

Lecture recordings

Classroom Interface (Main Screen)
Layout should include:

Top Navigation Bar:

Access Learn logo

Class name

Language selector dropdown

Leave class button

Main Teaching Area:

Shared slides or PDF viewer

Teacher video in floating bubble

Screen sharing capability

AI Transcript Panel:

Real-time lecture transcript

Speaker labels

Scrollable panel

Live Translation Panel:

Display translated subtitles

Option to show original + translated text

Right Sidebar:
Tabs for:

Chat

Students list

Assignments

Homework Submission Interface:

Upload image

Upload document

Show submitted assignments

Accessibility Section:

Small panel for Sign Language Interpreter (AI avatar)

General UI Goals:

The interface should feel like a futuristic AI classroom

Clear focus on accessibility and global learning

Simple navigation

Elegant typography

Minimal clutter
